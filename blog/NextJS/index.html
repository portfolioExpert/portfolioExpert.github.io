NextJS2022-10-22

NextJS 란

NextJS를 학습한 것을 정리하고 사용하는 이유를 알아봅니다.

Next.js

1. NextJS 란?

  • 공식 사이트: nextjs.org
  • 생산용 React 프레임워크
  • 대규모의 양산형 React 앱을 더 편리하게 구축할 수 있도록 많은 기능을 제공
  • ReactJS의 풀스택 프레임워크

2. 주요 기능

1. SSR 기능으로 SEO 향상

  • 콘텐츠가 많이 있는 공개 페이지인 경 검색 엔진 최적화가 중요
  • 서버에서 해당 페이지를 사전 렌더링한 상태에서 서버에 요청이 들어 왔을 때 화면, 컴포넌트를 미리 렌더링 시켜 놓는다.
  • nextjs를 적용하면 첫 페이지 로딩 이후 SPA를 적용시킬 수 있다.

2. 파일 기반 라우팅으로 라우팅 간소화

  • 동적 라우팅, 중첩 라우팅을 지원한다.
  • SPA에서 적용했던 라우터 라이브러리를 nextjs로 적용시키는 것
  • HTML에서 여러개의 html을 사용해 페이지를 연결하는 것 처럼 라우팅 개념을 사용한다.

3. NextJS는 풀스택 프레임워크

  • React에 백엔드 코드를 추가할 수 있다.
  • 하나의 파일 내에서 nodeJS 문법으로 db연동과 API를 request할 수 있다.

3. 프로젝트 만들기

$ npx create-next-app

$ npm install
  • Create-next-app을 통해 프로젝트를 생성
  • npm install을 통해 package.json의 모듈을 node_modules로 생성
  • index.html이 없는 이유는 NextJS는 사전 렌더링 기능을 내장하기 때문이며 서버에 요청이 오면 이 싱글 페이지에 동적으로 사전 렌더링을 거 콘텐츠를 포함한 초기 페이지를 보여주게 됩니다.

생성시 폴더 구조

-pages
-public
-styles
  • pages: 파일 기반 라우팅을 설정하고 애플리케이션을 구성할 여러 페이지를 정의

4. NextJS에서 파일 렌더링

  • 하단에 폴더를 만들고 index.js를 두면 /폴더이름 호출시 라우팅 된다.

5. 동적 페이지 추가 기능(동적 라우팅)

  • 파일 이름에 [ ]를 넣고 그 안에 식별자를 넣어준다.
  • 그러면 기본 url 뒤에 여러 식별자가 붙어올 수 있다.

1. 동적 매개변수 값 추출하기

// our-domain.com/news
import { useRouter } from "next/router";
function DetailPage() {
  const router = useRouter();

  //대괄호에 넣었던 변수와 같은 변수를 query에 넣어준다
  const newsId = router.query.newsId;

  return <h1>DetailPage</h1>;
}
export default DetailPage;
  • useRouter로 라우터 객체에 접근하고 라우터 객체에서 특정 데이터나 호출할 수 있는 특정 메소드를 얻을 수 있다.
  • 또한 query를 통해 동적 경로 세그먼트의 구체적인 값에 접근할 수 있다.
  • 라우터는 페이지를 처음 렌더링할 때 즉시 실행되는데 이 시점에서는 URL이 무엇인지 아직 모릅니다.
  • 하지만 정보가 들어오면 다시 컴포넌트를 렌더링 시키고 구체적인 값을 얻는다.

2. NextJS에서 SPA로 페이지 전환 시키기

  • 를 이용해 다른 링크로 넘어갈 때 새로고침 아이콘이 잠깐 X로 떳다가 새로고침 아이콘으로 돌아온다면 브라우저가 새 요청을 보내고 HTML 페이지를 받는다는 신호 입니다.
  • 이때 상태를 관리하는 redux의 상태 context 상태 등이 없어질 수 있습니다.
// our-domain.com/news
import {Fragment} from 'react';
import Link from 'next/link';

function NewsPage() {
  return (
  <Fragment>
    <h1>NewsPage</h1>
    <ul>
      <li>
        <Link href="/news/nextjs-is-a-great-framework">
          NextJS Is A Great Framework
          </Link>
        </li>
        <li>
          Something Else
          </li>
    </ul>
    </Fragment>
    )
}
export default NewsPage;
  • Link 태그를 통해 NextJS위에서 SPA를 적용 시키고 새로운 HTML을 받아오지 못하도록 해주며 불러올 컴포넌트를 읽어 들이고 URL을 변경시키고 페이지가 바뀐 것 처럼 보이게 합니다.
